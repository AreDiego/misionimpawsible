Créditos de imágenes

Las fotografías utilizadas en este sitio web fueron recuperadas únicamente con fines académicos para ilustrar el proyecto Misión ImPawsible.

Fotografías

La mayoría de las imágenes utilizadas fueron obtenidas de la página oficial de Facebook "Huellitas Felices Huichapan", con el único propósito de apoyar visualmente este proyecto académico y sin fines comerciales.

Aviso

Todos los derechos de las fotografías pertenecen a sus respectivos autores y administradores de la página. Su uso dentro de este sitio tiene únicamente fines educativos y de aprendizaje.

Elaboración del sitio web

Este sitio web fue desarrollado como parte del proyecto académico Misión ImPawsible, correspondiente al Módulo 22 de Prepa en Línea-SEP.

HTML5

Todo el sitio fue desarrollado utilizando HTML5 para estructurar la información y organizar cada una de las secciones del proyecto.

Canva

Los elementos gráficos, tarjetas, banners e imágenes personalizadas fueron diseñados utilizando Canva para brindar una mejor experiencia visual.

Proyecto académico

Todo el contenido, la investigación, la estructura del sitio y el desarrollo del proyecto fueron elaborados con fines exclusivamente académicos para Prepa en Línea-SEP.

Misión ImPawsible

Proyecto académico desarrollado para Prepa en Línea-SEP, enfocado en promover el bienestar animal, la adopción responsable y la participación de la comunidad.
